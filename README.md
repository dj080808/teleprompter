# Whisper 提词器

一个基于 Whisper 转录文件的 KTV 风格提词器应用，支持分段录音、预览和音频合并导出。

## 功能特点

- ✅ 解析 Whisper 转录文件（支持 `[时间-时间]` 格式）
- ✅ 自动/手动两种播放模式
- ✅ 上方居中的"..."变色提示（灰色→红色渐变）
- ✅ 大字体文案显示，类似 KTV 歌词效果
- ✅ 分段录音功能（44.1kHz 采样率）
- ✅ 支持重录任意句子
- ✅ 预览已录制音频
- ✅ 音频合并导出为 WAV 格式
- ✅ 快捷键支持

## 安装依赖

```bash
pip install sounddevice numpy
```

## 使用方法

1. 运行程序：
```bash
python teleprompter.py
```

2. 点击"📂 加载文件"按钮或程序会自动加载 `RA2_English_Colloquial.txt`

3. 选择模式：
   - **手动模式**：使用导航按钮或方向键手动切换句子
   - **自动模式**：按时间轴自动播放，模拟 KTV 效果

4. 录音：
   - 定位到要录制的句子
   - 点击"⏺ 开始录制"或按 `R` 键
   - 跟随提词器朗读文案
   - 点击"⏹ 停止录制"保存
   - 已录制的句子会显示 ✓ 标记

5. 预览：
   - 选择已录制的句子
   - 点击"🔊 预览"或按 `P` 键播放

6. 导出：
   - 录制完所有或部分句子后
   - 点击"💾 导出"或按 `Enter` 键
   - 选择保存位置，程序会自动合并所有片段

## 快捷键

- `空格` - 播放/暂停
- `←` / `→` - 上一句/下一句
- `R` - 开始/停止录制
- `P` - 预览当前句子
- `Enter` - 导出音频

## 文件格式

支持的转录文件格式：
```
[0.00-2.68]
Today, let's jump into a real mind-game match.

[2.68-4.96]
This is an eight-player free-for-all.
```

## 输出文件

- 录音片段保存在 `recordings/` 目录
- 最终导出的完整音频为 WAV 格式（44.1kHz，单声道）
